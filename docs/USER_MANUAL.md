# BAMCLauncher 用户使用手册

## 一、简介

BAMCLauncher是一款基于Flutter开发的跨平台Minecraft启动器，支持Windows、macOS和Linux系统。采用清新的方块风格设计，融合了Minecraft和蔚蓝档案的视觉元素，提供完整的游戏管理体验。

### 主要特性

- ✅ 跨平台支持：Windows、macOS、Linux全平台覆盖
- ✅ 多账户管理：支持微软正版、离线账户、第三方登录
- ✅ 全版本支持：正式版、快照版、远古版全覆盖
- ✅ 模组加载器：Forge、Fabric、Quilt、NeoForge自动安装
- ✅ 内容管理：模组、资源包、光影一键安装
- ✅ 整合包支持：CurseForge、Modrinth等全格式兼容
- ✅ 服务器管理：服务器列表、一键加入、局域网联机
- ✅ 自定义界面：清新方块风格，流畅动效

## 二、安装指南

### Windows系统

#### 方法一：安装包安装

1. 下载最新版本的 `BAMCLauncher-Setup.exe` 安装包
2. 双击运行安装程序
3. 按照安装向导完成安装
4. 安装完成后，桌面会生成快捷方式

#### 方法二：便携版使用

1. 下载 `BAMCLauncher-Portable.zip` 压缩包
2. 解压到任意位置
3. 运行 `BAMCLauncher.exe` 启动程序

### macOS系统

1. 下载最新版本的 `BAMCLauncher.dmg` 安装包
2. 双击打开dmg文件
3. 将 `BAMCLauncher` 拖到应用程序文件夹
4. 首次运行时，右键点击选择"打开"以允许运行

### Linux系统

#### 方法一：AppImage（推荐）

1. 下载 `BAMCLauncher-x86_64.AppImage`
2. 赋予执行权限：`chmod +x BAMCLauncher-x86_64.AppImage`
3. 双击运行

#### 方法二：Debian/Ubuntu

1. 下载 `bamclauncher_1.0.0_amd64.deb`
2. 安装：`sudo dpkg -i bamclauncher_1.0.0_amd64.deb`
3. 运行：`bamclauncher`

#### 方法三：Fedora/CentOS

1. 下载 `bamclauncher-1.0.0-1.x86_64.rpm`
2. 安装：`sudo rpm -ivh bamclauncher-1.0.0-1.x86_64.rpm`
3. 运行：`bamclauncher`

## 三、快速入门

### 第一次启动

1. 启动BAMCLauncher
2. 选择语言（支持中文、英文）
3. 选择游戏目录（默认自动创建）
4. 添加账户（微软账户或离线账户）
5. 下载并安装游戏版本
6. 点击"启动游戏"开始体验

#### 侧边栏功能说明

- **版本管理**：浏览、下载、安装游戏版本和模组加载器

## 四、功能使用指南

### 账户管理

#### 添加微软账户

1. 点击侧边栏"账户"选项
2. 点击"添加账户"按钮
3. 选择"微软账户"
4. 在弹出的浏览器中登录微软账户
5. 授权BAMCLauncher访问您的Minecraft账户
6. 登录成功后自动返回启动器

#### 添加离线账户

1. 点击侧边栏"账户"选项
2. 点击"添加账户"按钮
3. 选择"离线账户"
4. 输入用户名（可选：选择皮肤）
5. 点击"创建"完成添加

#### 切换账户

1. 在账户列表中选择要切换的账户
2. 点击"设为当前账户"按钮
3. 账户切换立即生效

### 版本管理

#### 安装游戏版本

1. 点击侧边栏"版本管理"选项
2. 点击"浏览版本"按钮
3. 在版本列表中选择要安装的版本
4. 点击"安装"按钮
5. 等待下载和安装完成

#### 安装模组加载器

1. 在版本管理页面，选择一个已安装的游戏版本
2. 点击"安装加载器"按钮
3. 选择加载器类型（Forge/Fabric/Quilt/NeoForge）
4. 选择加载器版本
5. 点击"安装"按钮
6. 等待安装完成

#### 创建自定义版本

1. 点击"创建自定义版本"按钮
2. 输入版本名称和ID
3. 选择继承的基础版本
4. 点击"创建"按钮

### 模组管理

#### 搜索并安装模组

1. 点击侧边栏"模组管理"选项
2. 在搜索框中输入模组名称
3. 选择模组来源（CurseForge/Modrinth）
4. 在搜索结果中找到想要的模组
5. 点击"安装"按钮
6. 选择要安装到的游戏版本
7. 等待下载和安装完成

#### 更新模组

1. 在"已安装"标签页查看已安装的模组
2. 点击"检查更新"按钮
3. 选择需要更新的模组
4. 点击"更新"按钮

#### 卸载模组

1. 在"已安装"标签页找到要卸载的模组
2. 右键点击模组，选择"卸载"
3. 确认卸载操作

### 整合包管理

#### 导入整合包

1. 点击侧边栏"整合包"选项
2. 点击"导入整合包"按钮
3. 选择整合包文件（.zip/.mrpack/.modpack等）
4. 等待解析和安装完成

#### 创建整合包

1. 点击"创建整合包"按钮
2. 选择要包含的游戏版本
3. 选择要包含的模组、资源包等内容
4. 设置整合包信息（名称、版本、作者等）
5. 点击"创建"按钮

#### 导出整合包

1. 在整合包列表中选择要导出的整合包
2. 点击"导出"按钮
3. 选择导出格式
4. 选择保存位置
5. 等待导出完成

### 服务器管理

#### 添加服务器

1. 点击侧边栏"服务器"选项
2. 点击"添加服务器"按钮
3. 输入服务器名称、地址和端口
4. 点击"保存"按钮

#### 连接服务器

1. 在服务器列表中选择要连接的服务器
2. 点击"加入服务器"按钮
3. 选择游戏版本
4. 点击"启动并加入"按钮

#### 局域网联机

1. 点击"开始局域网游戏"按钮
2. 选择世界名称和端口
3. 点击"开始"按钮
4. 其他玩家可以在多人游戏中看到您的局域网游戏

### 游戏启动

#### 快速启动

1. 在主页选择要启动的版本
2. 点击"启动游戏"按钮
3. 等待游戏启动

#### 自定义启动参数

1. 在版本管理页面，点击版本右侧的"设置"按钮
2. 配置JVM参数、游戏参数等
3. 保存设置后启动游戏

## 五、高级功能

### 配置管理

#### 修改游戏目录

1. 点击侧边栏"设置"选项
2. 在"游戏设置"中点击"浏览"按钮
3. 选择新的游戏目录
4. 点击"保存"按钮

#### 内存设置

1. 在"游戏设置"中找到"内存分配"
2. 设置最大内存（建议为系统内存的50%-75%）
3. 设置最小内存（可选）

#### Java路径设置

1. 在"游戏设置"中找到"Java设置"
2. 点击"浏览"按钮选择Java可执行文件
3. 或勾选"自动检测Java"让启动器自动查找

### 下载设置

#### 镜像源设置

1. 在"下载设置"中选择下载镜像源
2. 可选：官方、BMCLAPI、MCBBS等
3. 可以设置多个备用镜像源

#### 下载线程设置

1. 在"下载设置"中设置下载线程数
2. 根据网络情况调整（建议4-8线程）

### 界面设置

#### 主题设置

1. 在"界面设置"中选择主题
2. 支持浅色/深色主题切换

#### 语言设置

1. 在"界面设置"中选择语言
2. 支持中文、英文等多语言

## 六、界面元素说明

### 按钮样式

- **主按钮**：蓝色渐变背景，悬浮时有放大效果
- **次要按钮**：白色背景带边框，悬浮时有颜色变化
- **危险按钮**：红色背景，用于删除等危险操作
- **图标按钮**：仅显示图标，悬浮时显示提示文字

### 进度条

- 采用方块化设计，融合Minecraft经验条元素
- 支持进度动画和百分比显示

### 弹窗

- 毛玻璃背景效果，居中显示
- 支持拖拽移动（按住标题栏）
- 点击外部区域可关闭（部分弹窗）

### 右键菜单

- 支持在列表项上右键点击
- 提供快捷操作选项
- 支持键盘快捷键

## 八、使用技巧

### 优化游戏性能

1. 根据电脑配置合理设置内存分配
2. 关闭不必要的后台程序
3. 使用OptiFine等优化模组
4. 调整游戏视频设置（渲染距离、图形质量等）

### 快速切换版本

- 在主页可以快速切换最近使用的版本
- 使用版本收藏功能标记常用版本

### 模组冲突解决

1. 使用"检查冲突"功能检测模组冲突
2. 查看模组依赖关系，确保安装必要的依赖模组
3. 检查模组版本兼容性

### 整合包导入技巧

- 大型整合包建议使用有线网络下载
- 导入前确保有足够的磁盘空间
- 导入过程中不要中断，否则可能导致文件损坏

## 九、注意事项

1. **账户安全**：不要在公共电脑上保存账户信息
2. **文件备份**：定期备份游戏存档和重要配置
3. **版本兼容**：安装模组前确认与游戏版本兼容
4. **网络安全**：仅从可信来源下载模组和整合包
5. **权限要求**：某些操作可能需要管理员权限
